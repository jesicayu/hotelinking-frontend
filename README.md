## About The Project

<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Features

- Create users.
- Log in and log out users.
- View offers.
- Generate a coupon code for an offer.
- View a list of the offers that the user has created a coupon for.
- Redeem a coupon.

## Getting Started

### Prerequisites

In order to run this project, you need to first install:

- Node.js

### Installation

2. Clone the repo
   ```
   git clone https://github.com/jesicayu/hotelinking-frontend
   ```
3. Install NPM packages
   ```
   npm install
   ```
4. Start the app!
   ```
   npm run dev
   ```
5. The app will be running on:
   ```
   http://localhost:5173/
   ```
